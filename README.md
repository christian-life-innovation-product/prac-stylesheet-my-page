# CSS学習プロジェクト - 自分のホームページを作ろう

## 📚 プロジェクト概要

このプロジェクトは、CSSを段階的に学習するための教材です。HTMLの基本構造はすでに用意されているので、CSSを書くことに集中できます。

**学習目標:**
- CSSの基本的なプロパティを理解する
- レイアウトの作り方を学ぶ（Flexbox、Grid）
- レスポンシブデザインの基礎を身につける
- 実際に動くWebページを完成させる

## 🎯 完成イメージ

このプロジェクトを完成させると、以下の要素を持つ素敵なホームページが作れます：

1. **ヘッダー** - グラデーション背景の美しいタイトル
2. **画像ギャラリー** - グリッドレイアウトで配置された6枚の画像
3. **自己紹介セクション** - プロフィール画像と説明文
4. **フッター** - 著作権表示

## 🚀 始め方

### 1. ファイルを開く

- `src/index.html` - HTMLファイル（編集不要）
- `src/style.css` - CSSファイル（ここにコードを書きます）

### 2. ブラウザで確認

`index.html`をブラウザで開いて、変更を確認しながら進めましょう。

### 3. 各ステップを順番に進める

`style.css`には14のステップが用意されています。各ステップには：
- 学習内容の説明
- TODOコメント
- ヒント（コード例）

が含まれています。

## 📖 学習ステップ

### ステップ1: 基本設定とリセット
**学習内容:** `*`, `box-sizing`, `margin`, `padding`

すべての要素の余白をリセットし、ボックスモデルを統一します。

### ステップ2: フォントと基本スタイル
**学習内容:** `font-family`, `line-height`, `color`, `background-color`

ページ全体のフォント、行間、色を設定します。

### ステップ3: コンテナの設定
**学習内容:** `max-width`, `margin`（自動センタリング）, `padding`

コンテンツの最大幅を設定し、中央に配置します。

### ステップ4: ヘッダーのスタイリング
**学習内容:** `background`, `linear-gradient`, `text-align`, `padding`

ヘッダーに美しいグラデーション背景を追加します。

### ステップ5: 見出しのスタイリング
**学習内容:** `font-size`, `margin`, `font-weight`

各見出しのサイズと余白を調整します。

### ステップ6: ギャラリーグリッドレイアウト
**学習内容:** `display: grid`, `grid-template-columns`, `gap`

CSS Gridを使って画像を美しく配置します。

### ステップ7: ギャラリーアイテムのスタイリング
**学習内容:** `border-radius`, `box-shadow`, `transition`, `:hover`

カードデザインとホバーエフェクトを追加します。

### ステップ8: 画像プレースホルダーのスタイリング
**学習内容:** `display: flex`, `justify-content`, `align-items`

Flexboxを使って要素を中央に配置します。

### ステップ9: 自己紹介セクションのレイアウト
**学習内容:** `display: flex`, `gap`, `align-items`

横並びのレイアウトを作成します。

### ステップ10: プロフィール画像のスタイリング
**学習内容:** `width`, `border-radius`, `flex-shrink`

円形のプロフィール画像を作ります。

### ステップ11: リストのスタイリング
**学習内容:** `list-style`, `border-bottom`

リストマーカーを削除し、区切り線を追加します。

### ステップ12: フッターのスタイリング
**学習内容:** `background-color`, `text-align`

暗い背景のフッターを作ります。

### ステップ13: セクションの余白設定
**学習内容:** `padding`, `margin`

各セクション間の余白を調整します。

### ステップ14: アニメーションの追加（ボーナス）
**学習内容:** `@keyframes`, `animation`

ヘッダーにフェードインアニメーションを追加します。

## 💡 学習のコツ

1. **一つずつ進める**
   - 各ステップを順番に進めましょう
   - 完成したら必ずブラウザで確認

2. **ヒントを活用する**
   - わからない時はヒントのコードを参考にしましょう
   - コメントアウト (`/* */`) を外すだけで動作確認できます

3. **自分でカスタマイズする**
   - 色を変えてみる
   - サイズを調整してみる
   - 新しいプロパティを追加してみる

4. **検証ツールを使う**
   - ブラウザの開発者ツール（F12）を開いて、CSSの動作を確認しましょう

## 🎨 カスタマイズアイデア

基本のステップを完了したら、以下のカスタマイズに挑戦してみましょう：

- **色の変更:** グラデーションや背景色を好きな色に変える
- **フォントの追加:** Google Fontsから好きなフォントをインポート
- **新しいセクションの追加:** 趣味や特技のセクションを追加
- **アイコンの追加:** Font Awesomeを使ってアイコンを追加
- **実際の画像を追加:** プレースホルダーを実際の画像に置き換え
